## circleci
- 自動建化部署、建置、測試
- CI (持續整合)
- CD (持續部署)

#### install
```
cd /var/www/hello-world
mkdir .circleci
cp    tools/circleci/hello-world-config.yml  .circleci/config.yml
                     ^^^^^^^^^^^^^^^^^^^^^^
```

#### github
- 請將 github 的帳號連動到 circleci
- https://circleci.com/dashboard

#### pem file
- 請將 pem 檔加到 circleci
- https://circleci.com/dashboard
